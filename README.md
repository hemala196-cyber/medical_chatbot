# Medical Chatbot

A Python-based chatbot that provides basic information about common medicines.  
It uses simple Natural Language Processing (NLP) and text-to-speech (TTS) to interact with the user.

## Features
- Lookup basic information about common medicines
- Text-based input with TTS output
- Educational project for learning Python and NLP concepts

## Requirements
- Python 3.x
- Libraries:
  - scikit-learn
  - pyttsx3

Install the required libraries using:

```bash
pip install scikit-learn pyttsx3
## How to run the chotbot
python medical_chatbot.py

## Example

`Enter medicine name: dolo 650`
Output: Dolo 650 is used to reduce fever and relieve mild to moderate pain.

`Enter medicine name: amlodipine`
Output: Amlodipine is used to treat high blood pressure and chest pain.

`Enter medicine name: rantac`
Output: Rantac helps reduce acid production in the stomach.

## Disclaimer
This chatbot is for **educational purposes only**.  
It does **not replace medical advice** from a professional doctor.
